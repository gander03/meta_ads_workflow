# Meta Ad Workflow

A commissioned marketing automation web application for one-click batch Facebook ad creation and automated ad variation generation, built with React and Python Flask, featuring integration with the Meta Ads API to automate campaign, ad set, and ad management workflows.

## Features

- One-click batch Facebook ad creation
- Integration with the Meta Ads API for automated campaign and ad management
- Automated ad variation and creative generation
- Streamlined campaign, adset and ad creation
- Prioritisation of top-performing ads and creatives
- React frontend with JS/TS support
- Python backend powered by the Flask library
- Cross-platform support for Windows and macOS

## Tech Stack

### Frontend
- React
- JavaScript / TypeScript

### Backend
- Python
- Flask

## Overview

The application was designed to streamline large-scale advertising workflows by reducing the manual effort involved in creating and testing Facebook ad campaigns. It supports rapid generation of ad variations and simplifies the process of identifying high-performing creatives for campaign optimisation.
